做实验：直接claude code，非常方便，报告也让claude写。没有验收，所以代码根本不用看，最后运行完截个图放报告里。

期末复习：考前学长也像屏幕前的诸位一样，在各大平台苦苦搜索往年题和复习经验，可惜一无所获。据说这是研究生课下放，所以很难找资源，只能看老师的pdf。

我用gemini把老师的pdf按知识点汇总，并把每一章生成一个新的pdf，可读性和可理解性大大增加，阅读时仍需结合ai工具和老师的原版pdf。



考试A卷内容：以下为老师原话：考试涉及课堂讲授的所有章节，A\B巻覆盖重要知识点，一判断题10题20分，每章涉及3个左右的判断题，二工具积累与应用，两小题共10分，涉及重要工具的积累和应用；三简答题，两小题共10分，涉及重要概念和重要计算或证明；四算法设计，两小题共30分，涉及典型的算法设计过程；五算法分析，1小题共13分，渐进式运用工具完成算法的分析，得出结论；六知识运用，17分，运用课堂知识解决人工智能、数据管理等领域的典型问题。

内容回忆：大题考了 摸球放回c个，是鞅的证明，算法设计考了矩阵乘法验证结果正确，基本算法（就是n的三次方复杂度那个）和优化

最后两道大题有证明，有一个是用瓦尔德方程，有一个是azuma不等式，建议学这两个的时候，可以做一道老师pdf里的例题，知道定理怎么用。

复习建议：

预算2天复习：第一天按照pdf把所有知识点全过一遍，把重要定理都理解，并且理解证明思路（有的实在难的图论定理可以略过，图论似乎没怎么考），

第二天把pdf例题看了，有证明题也有设计算法的题，主要集中在23456章，还是实在难的看不懂就跳过。基本上两天就非常稳了，实际上我只复习了第一天，我没看例题，但是上考场也够用，最后两道证明题是没问题的，算法设计题会差的多。

大家根据自己的时间衡量就好，这课平时分60且是选修，一般都没事，仅仅给大家点复习经验，不会像我们当时那么慌。考试题目不会像作业那么难。
